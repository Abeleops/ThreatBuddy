## Overview

This project enhanced our web development and UI/UX design skills by applying HTML and CSS to create a clean, minimalist interface inspired by the FEU color palette. JavaScript was used to implement interactive features and improve the user experience, while PHP and MySQL were utilized to develop the backend, manage user accounts, and integrate the database for storing and retrieving information efficiently. Through this project, we gained practical experience in designing and developing a functional, database-driven web application.

### Key Components

- Admin Dashboard (CRUD)
- Hero Section
- Cyber Threat Details Page
- TMedia (Article Page)
- Search and Filter
- Database Integration
- Navigation Bar & Footer
- Categories/Tags
- Security Features (Input validation, session handling)
